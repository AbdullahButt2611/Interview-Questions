# Difference Between AI, ML, and DL

## In One Line

AI is the big goal of making machines act smart. ML is one way to reach it by learning from data. DL is a powerful kind of ML that uses deep neural networks. Each one lives inside the one before it: AI contains ML, and ML contains DL.

<br><br>

## The Big Picture

Think of three boxes, one inside the other. AI is the largest, ML is a part of AI, and DL is a part of ML.

```
+---------------------------------------------+
|  Artificial Intelligence (AI)               |
|  make machines act smart                    |
|                                             |
|   +-------------------------------------+   |
|   |  Machine Learning (ML)              |   |
|   |  learn patterns from data           |   |
|   |                                     |   |
|   |   +-----------------------------+   |   |
|   |   |  Deep Learning (DL)         |   |   |
|   |   |  deep neural networks       |   |   |
|   |   +-----------------------------+   |   |
|   |                                     |   |
|   +-------------------------------------+   |
|                                             |
+---------------------------------------------+
```

<br><br>

## A Simple Analogy

Russian nesting dolls. AI is the biggest doll. Open it and you find ML inside. Open ML and you find DL inside. They are not rivals, each is a smaller, more specific piece of the bigger one.

<br><br>

## Each One, Briefly

- **AI (Artificial Intelligence):** the whole field of making machines do things that normally need human smarts, like deciding, understanding language, or recognizing images. It can use fixed rules or learning.
- **ML (Machine Learning):** a part of AI where the machine learns patterns from data instead of being given fixed rules. The more good data it sees, the better it gets.
- **DL (Deep Learning):** a part of ML that uses deep neural networks (many stacked layers loosely inspired by the brain). It shines on messy data like images, audio, and text, and it figures out the useful features on its own.

<br><br>

## Side by Side

| | AI | ML | DL |
|---|---|---|---|
| What it is | making machines act smart | machines that learn from data | ML using deep neural networks |
| Scope | the whole field | a part of AI | a part of ML |
| How it works | rules, logic, or learning | finds patterns in data | learns through many layers |
| Picking features | depends | humans choose them | the model learns them itself |
| Data needed | varies | moderate | large |
| Examples | rule-based assistant, game AI | spam filter, recommendations | image recognition, chatbots, self-driving vision |

<br><br>

## Key Takeaway

They are not three competing things. They are nested: DL is a type of ML, and ML is a type of AI. Use AI for the broad goal, ML when the machine learns from data, and DL when that learning uses deep neural networks.
